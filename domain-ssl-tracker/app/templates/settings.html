Gmail Setup Guide

  1. Go to Google Account → Security
  2. Enable 2-Step Verification if not already on
  3. Go to App Passwords
  4. Select app: Mail → Select device: Other → name it "SSL Tracker"
  5. Copy the 16-character password and paste it in the SMTP Password field above
  6. Use your Gmail address as both SMTP Username and Send Alerts From
